
Neste projeto, eu me concentrei na utilização do Flexbox, uma poderosa técnica de layout em CSS, para criar uma página da web dedicada a São Paulo. Aqui está um resumo mais detalhado:

Estrutura Geral

Utilizei HTML para estruturar a página, definir títulos, cabeçalho, corpo e rodapé.
Linkei um arquivo CSS externo chamado "style.css" para aplicar estilos à página.
Cabeçalho com Flexbox

No cabeçalho, criei um logotipo com uma imagem e uma barra de navegação com links.
Utilizei o Flexbox para organizar os elementos do cabeçalho de forma flexível e responsiva.
Defini o cabeçalho como um contêiner flexível com "justify-content" para distribuir espaços igualmente entre os elementos.
Barra de Navegação com Flexbox

A barra de navegação contém links para informações sobre São Paulo, como "Sobre", "Curiosidades" e "Cultura".
Usei o Flexbox para alinhar os itens horizontalmente e espaçá-los uniformemente.
Adicionei estilos para realçar os links quando o mouse passa sobre eles.
Conteúdo Principal com Flexbox

Na seção principal, adicionei imagens relacionadas a São Paulo.
Utilizei o Flexbox para criar um layout flexível e responsivo, organizando as imagens em uma grade.
Defini as imagens como itens flexíveis, permitindo que elas se ajustem automaticamente ao tamanho da tela.
Efeitos de Imagem com Hover

Ao passar o mouse sobre as imagens, apliquei um efeito de escala de cinza usando CSS.
Isso foi feito com facilidade graças ao Flexbox, que garantiu a disposição das imagens em linha.
Rodapé Simples

No rodapé, adicionei um texto simples com informações sobre a turma "Elas na tech 2023.Turma 7".
Estilos Gerais

Apliquei estilos gerais ao corpo da página, definindo a fonte e o plano de fundo.
Defini estilos para o cabeçalho, como cores de fundo, espaçamento e cores de texto.
Estilizei a barra de navegação para que os links fiquem destacados ao passar o mouse.
Organizei as imagens em uma grade flexível com efeitos de escala de cinza ao passar o mouse.
Estilizei o rodapé com cores de fundo e texto.
Flexbox é a Estrela

O destaque deste projeto é a utilização do Flexbox para criar layouts flexíveis e responsivos, tornando a página agradável em qualquer tamanho de tela.
Com Flexbox, conseguimos alinhar elementos, distribuir espaços uniformemente e criar uma grade de imagens com facilidade.
Espero que este projeto sirva como um ótimo exemplo de como o Flexbox pode ser poderoso no desenvolvimento web. É uma técnica essencial para criar layouts atraentes e responsivos.
